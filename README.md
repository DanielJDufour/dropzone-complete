# dropzone-complete
Dropzone Web Component Complete with Display of GeoTIFF, JPG, and PNG Files!

# install
```bash
npm install dropzone-complete
```

# usage
```html
<script src="https://unpkg.com/dropzone-complete"></script>

<dropzone-complete></dropzone-complete>
```
